# XiaoMI-Shop

小米商城前端仿站实训项目，用于完成一周前端页面开发练习。项目采用原生 HTML、CSS 和 JavaScript 编写，重点训练静态页面结构搭建、公共样式复用、页面跳转和基础交互组织。

## 项目说明

本项目只做前端静态页面展示和基础 JavaScript 交互，不连接后端数据库，不做真实登录、订单和支付功能。

## 功能范围

- 首页：首页入口、导航栏和基础页面框架。
- 商品列表页：展示商品列表内容的页面入口。
- 商品详情页：展示单个商品信息的页面入口。
- 购物车页：展示购物车内容和后续结算流程的页面入口。
- 登录页：展示账号登录表单的页面入口。
- 公共导航：各页面统一顶部导航和底部信息。

## 页面入口

- index.html：首页
- products.html：商品列表页
- detail.html：商品详情页
- cart.html：购物车页
- login.html：登录页
- css/style.css：公共样式
- js/main.js：公共脚本
- images/：图片素材

## 运行方式

项目为纯静态页面，不需要安装依赖。

1. 直接双击 `index.html` 在浏览器中打开。
2. 或使用 VS Code 的 Live Server 插件打开项目，访问首页后通过导航切换页面。

## 开发约定

- 公共样式统一写入 `css/style.css`。
- 公共脚本统一写入 `js/main.js`。
- 各页面保持相同的导航栏和页脚结构，方便后期整合。
- 页面内容开发完成后，优先检查导航跳转、移动端宽度和基础排版。
