# 小米商城小组开发说明

本项目是 Web 前端基础实训作业，只使用 HTML、CSS 和 JavaScript。

## 一、项目要求

- 每个页面都能直接在浏览器中打开。
- 页面之间的导航链接要能正常跳转。
- 不需要连接数据库。
- 不需要实现真实登录、支付和订单提交。
- 不使用 Vue、React 等框架。

## 三、开发时注意

- 修改前先确认自己负责哪个页面。
- `css/common.css` 由 A 张奕扬统一维护；其他成员需要修改公共样式时，先和 A 沟通，避免影响全部页面。
- 各成员主要修改自己负责的页面 CSS，不把页面专用样式写进 `css/common.css`。
- 公共导航和页脚保持一致。
- 图片统一放在 `images` 文件夹。
- 公共 CSS 写在 `css/common.css`，页面样式写在对应的 CSS 文件中；JS 写在 `js` 文件夹中。
- 不随便删除别人写好的页面和图片。
- 提交前先打开页面看一遍。

## 四、页面检查

完成后主要检查这些内容：

- 首页、商品页、详情页、购物车、登录页都能打开。
- 顶部导航能正常跳转。
- 商品图片能显示。
- 按钮文字没有超出。
- 手机宽度下不要出现明显重叠。
- 浏览器控制台没有明显报错。

```text
完成购物车数量加减和总价计算
修改登录页表单样式
```
